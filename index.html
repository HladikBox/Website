欢迎使用AppLink

一个面向开发者，解决数据问题的PaaS平台

AppLink自动为开发者生成一个可自由定制化的数据管理中心。减少开发者在开发移动互联网项目时，设计并编写数据管理中心的工作量，可以达到在最短时间内让开发者立即提供一个数据管理中心让客户录入数据。目的：改善用户体验，让用户觉得效率非常高。减少开发时候的工作量，不用编写一句代码就完成数据中心的构建。

AppLink将互联网的资源服务化为可编程接口，为第三方开发者提供有商业价值的资源和服务平台。有了AppLink平台的支撑，开发者在应用市场中可以获得大量的可二次开发的应用素材，这就为开发带来了极大的方便，不但提高了开发效率，还节约了开发成本。有了AppLink平台的支持，WEB应用的开发变得更加敏捷，能够快速响应用户需求的开发能力，也为最终用户带来了实实在在的利益。

接口 语言

上传最终完成的源代码是为了能够将你的项目分享，让别的开发者直接使用你的代码。

  • AJAX

    适用于网站，APP网站，微信，Cordova，Ionic v1等纯AJAX开发的网站

  • PHP

    适用于使用PHP开发，生成MVC中的Model的php代码，php页面直接调用。

  • Android JAVA

    下载Android原生方式的APP开发使用的Java接口代码，开发者可以直接在这个接口进行自己的业务流程开发。

  • Swift

    下载iPhone的原生APP开发使用的接口代码，开发者可以直接在这个接口进行自己的业务流程开发。

  • TypeScript

    适用于使用TypeScript进行开发的网站或者APP，例如Ionic v2。

  • C#

    适用于C#的程序开发，引用工程文件调用model对象。

  • Object-C

    下载iPhone的原生APP开发使用的接口代码，开发者可以直接在这个接口进行自己的业务流程开发。
